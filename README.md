# 🎯 Exercícios de Dart
 
Conjunto de exercícios práticos desenvolvidos em **Dart** com o objetivo de consolidar o aprendizado da linguagem, cobrindo desde os fundamentos até conceitos mais avançados, construindo uma base sólida para o desenvolvimento de aplicativos mobile.
 
---
 
## 🚀 Como Executar os Exercícios
 
### Pré-requisitos
 
- [Dart SDK](https://dart.dev/get-dart) instalado na sua máquina
- Editor de código (recomendado: [VS Code](https://code.visualstudio.com/) com a extensão **Dart**)
 
### Rodando um arquivo
 
```bash
dart run nome_do_arquivo.dart
```
 
Exemplo:
 
```bash
dart run Lista_de_Exercícios/variaveis.dart
```

## 🛠️ Tecnologias
 
- **Linguagem:** Dart
- **SDK:** Dart SDK (versão estável mais recente)
